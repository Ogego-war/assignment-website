My Projects

Web 2.0 Throwback Jukebox

Remember when the internet was all about rounded corners, gradients, and sharing EVERYTHING? The Web 2.0 Throwback Jukebox is a blast from the past, built to bring back the glory days of online music. I used HTML/CSS and JavaScript to create a retro-styled interface, complete with a spinning CD animation and a playlist that's pure 2012 indie rock (think Arcade Fire, The Black Keys, maybe a little Gotye). For the backend, I tapped into the power of Node.js and React to handle user interactions, playlist management, and even a 'suggest a song' feature. It's like a MySpace music player, but way less clunky! My role? I was the architect of this time machine, making sure the nostalgia flowed smoothly through every pixel.

Pixel Perfect Portfolio 2.0

Back in 2012, having a killer online portfolio was everything for a web developer. Pixel Perfect Portfolio 2.0 is my take on that, a site built with pure HTML/CSS and JavaScript to showcase my design and coding chops. Forget fancy frameworks, this is all about clean, hand-crafted code. I focused on creating a visually engaging layout with smooth animations and a user-friendly experience. Think image sliders with those classic drop shadows, maybe a little bit of early responsive design (because smartphones were starting to get big!), and a contact form that actually works. My role was to channel my inner web designer and front-end developer, bringing a static design to life with interactive elements. It's a tribute to the days when every pixel mattered and web development felt like a true craft.
